Правила игры Алиас

Цель игры

Объяснить как можно больше слов своей команде за отведенное время, не используя однокоренные слова и перевод на другие языки.

Подготовка

  1. Разделитесь на команды (2-4 команды)
  2. Выберите категории слов для игры
  3. Установите время раунда (60-180 секунд)
  4. Определите количество слов в раунде

Ход игры

  1. Объясняющий смотрит на слово на экране
  2. Объясняет его своей команде без использования однокоренных слов
  3. Если команда угадывает - нажимается "Угадано" (+1 очко)
  4. Если слово сложное - можно пропустить (-1 очко при включенном штрафе)
  5. При нарушении правил (однокоренное слово) - нажимается "Нарушение" (-2 очка)
  6. По истечении времени устройство передается следующей команде

Советы

  • Используйте синонимы, антонимы, ассоциации
  • Можно описывать внешний вид, функцию, местонахождение
  • Разрешается использовать жесты и мимику
  • Можно говорить "это похоже на..." или "это противоположность..."
  • Запрещены переводы на другие языки и однокоренные слова
